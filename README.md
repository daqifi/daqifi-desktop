# DAQifi Desktop

This is a windows (.NET) application that is used to comminicate with DAQifi hardware.

## Tech Stack

- .NET 4.5
- WPF

## Dependnecies

- EntityFramework (ORM)
- Goole Protocol Buffers (read incoming data from DAQifi hardware)
- MahApps (UI components)
- Oxyplot (for graphing)
